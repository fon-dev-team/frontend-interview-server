## Description

[Nest](https://github.com/nestjs/nest) server made to be used in frontend interviews.

## Installation

```bash
$ npm install
```

**NOTE: To run this project you must have docker and docker compose installed**

## Starting server

```bash
$ docker-compose up 
```

## Commands

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# incremental rebuild (webpack)
$ npm run webpack
$ npm run start:hmr

# production mode
$ npm run start:prod
```
